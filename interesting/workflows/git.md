# Git

## Installation

### MacOS

Install with HomeBrew

```
brew install git
```

## Usage

Clone

```
gh repo clone Arno1235/...
```

Push change

```
git add .
git commit -m <commit-mmesage>
git push
```

Git switch branch

```
git fetch origin
git checkout <branch-name>
```

Git pull rebase

```
git pull --rebase
```

Git force pull

```
git fetch --all
git reset --hard origin/master
```

Git force push

```
git push --force
```
